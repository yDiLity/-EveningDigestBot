# Daily Digest Bot (Бот «Итоги дня»)

Telegram-бот для сбора ежедневных отчетов от участников команды.

## Возможности

- Сбор отчетов в личных сообщениях
- Автоматическая публикация в группу в 21:00
- Мгновенный черновик после каждого сообщения
- Сравнение метрик между участниками
- Команды управления

## Установка

1. Клонируйте репозиторий
2. Установите зависимости:
```bash
pip install -r requirements.txt
```

3. Скопируйте `.env.example` в `.env` и заполните:
```bash
cp .env.example .env
```

## Настройка .env

| Переменная | Описание | Пример |
|-----------|----------|--------|
| BOT_TOKEN | Токен бота от @BotFather | 1234567890:ABC... |
| GROUP_CHAT_ID | ID группы для публикации | -1001234567890 |
| ADMIN_IDS | ID админов через запятую | 123456789,987654321 |
| TIMEZONE | Временная зона | Europe/Moscow |

## Запуск

```bash
python daily_digest_bot.py
```

## Команды

| Команда | Описание |
|---------|----------|
| /start | Настройка бота (при первом запуске) |
| /mydigest | Посмотреть текущий черновик |
| /sendnow | Отправить отчет немедленно |
| /undo | Удалить последнее действие |
| /skip | Пропустить сегодня |
| /leaderboard | Таблица лидеров по шагам |
| /timezone | Запустить голосование за время |

## Использование

1. Запустите бота командой `/start`
2. Бот спросит ваше имя для отображения
3. Пишите боту что сделали за день:
   - «прошел 5000 шагов»
   - «выпил 2 литра воды»
   - «съел борщ на обед»
   - «прочитал 30 страниц»
4. Бот показывает черновик после каждого сообщения
5. В 21:00 бот публикует все отчеты в группу

## Примеры сообщений

```
прошел 5000 шагов     → 🚶‍♂️ Шаги: 5,000
выпил 1.5 литра     → 💧 Вода: 1.5 л
съел овсянку        → 🍳 Еда: съел овсянку
прочитал 20 страниц → 📖 Страницы: 20
```
